# Vishnu Shah - Portfolio Website

This is my personal portfolio website showcasing my skills, projects, and contact information.  
Built with **React** and **Tailwind CSS** for a modern, responsive, and clean design.

## Features

- Smooth scrolling navigation
- Responsive design (desktop & mobile friendly)
- Projects showcase with links to GitHub repositories
- Contact form integration
- Dark / Light theme ready

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Icons & Animations:** React Icons, CSS animations
- **Hosting:** GitHub Pages / Netlify (optional)
